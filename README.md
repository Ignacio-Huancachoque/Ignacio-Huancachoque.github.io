# Mi Biografía Personal

Este es un proyecto de página web personal que muestra mi biografía, incluyendo una foto personal, descripción, formación académica, cursos cursados, enlaces a compañeros y datos de contacto.

## Estructura del Proyecto

- `index.html` - Página principal con la biografía
- `estilos.css` - Hoja de estilos con diseño en color verde agua
- `mi-foto.jpg` - Tu foto de perfil (debes agregar este archivo)
- `README.md` - Este archivo

## Características

- Diseño responsivo que se adapta a diferentes dispositivos
- Estética en tonos de verde agua
- Sección de perfil con foto y descripción personal
- Secciones organizadas para:
  - Formación académica (universidad y carrera)
  - Cursos cursados (7 elementos)
  - Enlaces a perfiles de LinkedIn de compañeros
  - Información de contacto (LinkedIn y email)

## Personalización

Para personalizar esta página con tu información:

1. Reemplaza la imagen `mi-foto.jpg` con tu propia foto
2. Actualiza la descripción en la sección "Sobre Mí"
3. Reemplaza los enlaces de la universidad y carrera por los tuyos
4. Actualiza la lista de cursos con los que has cursado
5. Agrega los enlaces de LinkedIn de tus compañeros
6. Coloca tu propio perfil de LinkedIn y dirección de email

## Tecnologías Utilizadas

- HTML5
- CSS3
